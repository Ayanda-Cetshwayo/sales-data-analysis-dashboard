# Sales Data Analysis Dashboard

## Overview

This project demonstrates an end-to-end data analysis workflow, transforming raw business data into meaningful insights to support decision-making.

## Objectives

* Clean and prepare raw data
* Analyse trends and performance
* Extract insights using SQL
* Build a visual dashboard

## Tools Used

* SQL
* Microsoft Excel
* Power BI

## Key Insights

* Top-performing products contributed the majority of revenue
* Sales showed consistent growth over time
* Certain regions underperformed and require attention
* Seasonal trends impacted overall performance
